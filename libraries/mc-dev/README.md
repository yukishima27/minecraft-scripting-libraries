> NOTE: This project is still under heavy development. Features may be removed or changed in future releases.

- [@minecraft/server](https://www.npmjs.com/package/@minecraft/server)

## Dependencies

- [@minecraft/math](https://www.npmjs.com/package/@minecraft/math)
- [@lpsmods/mc-utils](https://www.npmjs.com/package/@lpsmods/mc-utils)

## Features

- Includes a command (`mcdev:dev show`) to open developer tools UI.
- Offers block info tool displaying block data in the action bar.
- Features player info tool showing player location, rotation, and properties.
- Contains item info tool for displaying held item details.
- Supports debug mode for additional debugging information.
- Tools are configurable and can be enabled/disabled via settings.
- Integrates with entity and chunk events for real-time data updates.
